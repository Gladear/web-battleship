Règle du jeu :

Les joueurs choisissent la position de leurs bateaux.
Ils ne pourront pas la changer plus tard.
Une fois que les bateaux sont placés, le jeu se déroule en tour par tour.
Chacun leur tour, les joueurs chosissent un endroit (une case) où tirer sur le plateau de l'adversaire.
S'ils touchent un bateau, ils recoivent une indication 'touché', sinon c'est 'dans l'eau'.
Pour gagner il faut détruire tous les bateaux adverses.
